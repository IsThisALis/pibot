# 🤖 PiBot

🛩️ Бот для модерации групп в телеграме с интерактивными ответами.

## ⚙️ Функции

- **Ответы на фразы** — бот отвечает на определённые фразы
- **Команды для модерации** — `$kick`, `$ban`, `$mute`, `$unmute` с разными уровнями доступа
- **NUKE** — `$nuke n` удаляет последние `n` сообщений
- **Алиасы для команд** — синонимы в `synonyms.json` (например, `$nuke` также `$burn`)
- **Система доступа** — суперюзеры, админы, и обычные участники
- **Память** — бот переносит перезапуски с помощью PicklePersistence

## 🚀 Начало работы

```bash
git clone https://github.com/deltashrimp/pibot.git
cd pibot
./setup.sh
# edit dev/telegram-token — токен телеграм бота
# edit datafiles/superusers.json — спиоск суперюзеров бота
./launchbot.sh
```
### ➕️ Создание нового бота

**ПЕРЕД ВЫПОЛНЕНИЕМ `./launchbot.sh`**

Создайте нового бота в botfather через `/newbot`.

После создания и настройки бота, botfather выдаст токен. Cкопируйте токен в файл telegram-token.

### ➕️ Добавление бота в чат
Найдите своего бота в телеграме и добавьте его в свой чат.

## 🛠️ Команды

| Команда | Уровень доступа |
|---------|-----------------|
| `$nuke n` | админ группы | 
| `$kick @user` | суперюзер | 
| `$ban @user` | суперюзер | 
| `$mute @user` | админ группы | 
| `$unmute @user` | админ группы |

Команду можно исполнить **ответом** на сообщение или **@упоминанем**.

## 🌳 Структура проекта

```
pibot/
├── source/code.py              # основной код
├── datafiles/
│   ├── synonyms.json           # алиасы команд
│   ├── public-phrases.json     # пример фраз, на которые бот будет отвечать
│   └── public-superusers.json  # список суперюзеров
├── dev/
│   └── public-botinfo.md       # пример информации о боте
├── launchbot.sh                # запуск бота
└── setup.sh                    # скрипт установки
```
_setup.sh добавляет приватные версии публичных файлов_
