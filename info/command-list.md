Владелец (ранг 1)
пибот кинь в гулаг @user - Забанить пользователя
пибот верни @user - Разбанить пользователя
пибот ии - Переключить AI-бэкенд (Gemini / Groq)
пибот ранг n для @user - Изменить ранг пользователя (n = 2, 3, 4)

Admin+ (ранг 2)
пибот сотри n - Удаляет n+1 последних сообщений
пибот кикни @user - Выгнать пользователя

Admin (ранг 3)
пибот мут @user [n] - Замутить пользователя (на n минут, минимум 0.5)
пибот размут @user - Размутить пользователя

Все (ранг 4)
пибот био - Показать информацию о боте
пибот обновы - Показать список изменений
пибот команды - Показать этот список
пибот ранги - Список пользователей с особыми рангами
пибот инфа - Случайное число от 0 до 100

RP-команды
обнять, погладить, поцеловать и т.д. (ответом на сообщение)
