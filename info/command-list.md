Команды вызываются с префиксом `$`.

⚙️ Админ-команды
`$nuke N` - Удаляет N последних сообщений в чате
`$mute [@user] [N]` - Замутить пользователя (на N минут, если указано)
`$unmute [@user]` - Размутить пользователя

🦸 Суперюзер-команды
`$kick @user` - Выгнать пользователя
`$ban @user` - Забанить пользователя
`$changeai` - Переключить AI-бэкенд (Gemini / Groq)

🌹 RP-команды
обнять, погладить, похвалить, обматерить и т.д. 


👤 Служебные фразы
`пибот инфа` - Случайный ответ
`пибот био` - Показать информацию о боте
`пибот обновы` - Показать список изменений
`пибот команды` - Показать этот список

🤖 AI-функции
- Упоминание бота (через @) в ответе на любое сообщение
- Контекст из последних сообщений чата передаётся AI
